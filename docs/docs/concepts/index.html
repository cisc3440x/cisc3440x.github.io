    What is Machine Learning #

    “Field of study that gives computers the ability to learn without explicitly being programmed.”

    Arthur Samuel, 1959

    Arthur Samuel wrote a machine learning algorithm in 1959 to teach a computer to play checkers. His plan was to have the computer play 1000’s of games of checkers against itself to LEARN what the better POSITIONS were.

    In general the more games the computer could play the more it would learn. If it only played 10 games it would learn much less than play 10000.

    THE MORE YOU GIVE A LEARNING ALGORITHM THE OPPORTUNITY TO LEARN THE BETTER IT WILL PERFORM!!

    There are several major Categories of learning algorithms

    • supervised learning (Most Common in Real World applications)
    • unsupervised learning
    • recommender systems
    • reinforcement learning

    Supervised Learning Algorithms #

    Maps inputs to outputs #

    • input => output
    • x => y

    Supervised Machine Learning refers to algorithms that learn Mappings from input to output. In other words given an x what is the y. Given the input what is the ouput.

    Train on Labeled Data #

    You train the learning algorithm by feeding it labeled data. Labeled data is a data set that includes correct examples of x => y mappings.

    Applications #

    In all of the following application we would first train our models with examples of input x to correct output y mappings. After the model has been trained it can then take a brand new input x it has never seen before and try to predict the corresponding output y.

    Email spam filters #

    Input(x) Output(y)
    email spam?(0,1)

    If the input x is an email and the output y is labeled as either

    • spam
    • not spam.

    speech recognition #

    Input(x) Output(y)
    audio clip text

    If the input x is audio and the output is a printed string of text.

    Machine Translation #

    Online Advertising #

    Input(x) Output(y)
    ad, user info Click?(0,1)

    The algorithm will try and predict, given some info about a user wether they will.

    Input(x) Output(y)

    Visual Inspection in manufacturing #

    Input(x) Output(y)
