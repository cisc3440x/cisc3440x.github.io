    What is Machine Learning #

    “Field of study that gives computers the ability to learn without explicitly being programmed.”

    Arthur Samuel, 1959

    Arthur Samuel wrote a machine learning algorithm in 1959 to teach a computer to play checkers. His plan was to have the computer play 1000’s of games of checkers against itself to LEARN what the better POSITIONS were.

    In general the more games the computer could play the more it would learn. If it only played 10 games it would learn much less than play 10000.

    THE MORE YOU GIVE A LEARNING ALGORITHM THE OPPORTUNITY TO LEARN THE BETTER IT WILL PERFORM!!
